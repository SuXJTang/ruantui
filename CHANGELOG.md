## [0.1.0] — 2025-05-27

### 新增

#### 🍃 银杏叶交互演示页面

- **初始版本**：实现基于 PNG/SVG 的银杏叶飘落动画，包含三层视差（近景/中景/远景）、鼠标交互扰动、季节渐变背景、响应式布局（`2376a36`）
- **拖尾特效**：为飘落的银杏叶添加精美拖尾效果，通过 `requestAnimationFrame` 循环实时追踪叶子位置并生成 SVG 残影，支持根据叶子颜色动态渲染拖尾色彩（`32359ad`）
- **拖尾可见度优化**：提高拖尾不透明度（0.4→0.7），增加 `mix-blend-mode: screen` 和 `brightness(1.2)` 增强色彩；添加鼠标点击波纹动效和页面震动效果；受鼠标影响的叶子拖尾更长、颜色更亮（`26e197c`）

#### 📘 软件推荐系统全套文档

- **需求分析文档**：定义 4 类目标用户（普通/专业/技术爱好者/企业）、网站+APP 核心功能、非功能需求、盈利模式、风险评估、成功 KPI
- **系统架构设计**：前后端分离微服务架构（React + React Native + Node.js + PostgreSQL），7 个核心服务拆分，Docker/K8s 部署方案
- **UI 设计指南**：品牌色 `#3366FF` / `#FF9900`，12 栏网格系统，1200px 最大宽度，组件规范（导航栏、软件卡片、评分、表单等），暗色模式，WCAG 2.0 无障碍标准
- **开发流程**：41 个阶段从立项到发布，5 个里程碑计划（共 20 周），8 个角色职责定义

#### 🎨 网站概念原型

- **首页原型**：完整单页 HTML 原型，含导航栏、英雄区、分类展示、推荐软件卡片、排行榜、专业评测、用户评价、页脚等模块
- **交互脚本**：防抖/节流、暗色模式切换、骨架屏加载、`IntersectionObserver` 懒加载、`PerformanceObserver` LCP 监控
- **样式系统**：~2200 行 CSS，CSS 变量主题系统，完整暗色模式覆盖，移动端底部导航栏适配，`safe-area-inset` 支持

#### 🖼️ 静态素材

- 三款银杏叶 SVG 矢量图（黄色、白色、绿色）（`2376a36`）

### 优化

- **性能大幅提升**：CSS 选择器从后代选择器优化为属性选择器；`transform: translate3d()` / `translateZ(0)` 启用 GPU 硬件加速；`translate` 替代 `top/left` 减少重排；CSS 变量统一管理主题色和动画参数（`daa7b7b`）
- **内存占用优化**：`Map` 替代 `Object` 管理叶子位置数据；SVG 内容缓存避免重复解析；使用 `requestAnimationFrame` 限制更新频率（`daa7b7b`）
- **自适应性能模式**：根据 `navigator.hardwareConcurrency` 和移动端检测自动降级（叶子数 35→15，拖尾长度 12→5）；支持 `?performance=high|medium|low&debug=true` URL 参数手动控制（`daa7b7b`）
- **页面可见性 API**：页面不可见时暂停动画，恢复时重新初始化，避免后台标签页消耗资源（`daa7b7b`）
- **调试工具**：内置 FPS/叶子数/拖尾数性能监视器（`?debug=true` 开启），低 FPS 时自动降级动画参数（`26e197c`、`daa7b7b`）
- **拖尾系统重构**：`fixed` 定位替代 `absolute` 解决定位漂移；添加 `trail-fade` 动画；清除拖尾时增加 `fadeOut` 过渡；拖尾长度从 5 增加到 12，鼠标影响区域拖尾长度 18（`32359ad`）
- **鼠标交互增强**：添加点击波纹效果（300px 扩散半径）+ 页面震动反馈；鼠标影响半径 150px，强烈点击效果半径扩大至 225px（`32359ad`、`26e197c`）
- **响应式优化**：移动端隐藏远景层叶子减少性能开销（`daa7b7b`）
- **卡片样式升级**：`backdrop-filter: blur(10px)` 毛玻璃效果；CSS 变量驱动；悬停 `translateY(-12px)` + 顶部渐变条动效

### 修复

- 修复动画初始化时卡片过早显示的问题，增加 300ms 延迟确保叶子先行渲染（`32359ad`）
- 修复拖尾在叶子静止时产生过多冗余点的问题，添加 `distance > 3px` 阈值过滤（`32359ad`）
- 修复页面从后台切回时动画状态不一致，增加 500ms 延迟重建拖尾（`26e197c`）

## 版本格式

```

```

- **主版本号**：不兼容的 API 修改或重大架构重构
- **次版本号**：向下兼容的功能新增
- **修订号**：向下兼容的问题修正
